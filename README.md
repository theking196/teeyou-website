# Teeyou Hair Website

A Next.js 14 website with App Router and Tailwind CSS.

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

## Project Structure

```
teeyou-website/
├── app/                    # Next.js App Router
│   ├── layout.js          # Root layout
│   ├── page.js            # Home page
│   └── globals.css        # Global styles
├── public/                 # Static assets
├── src/
│   └── components/        # Reusable components
├── package.json
├── tailwind.config.js
├── postcss.config.js
└── next.config.js
```

## Tech Stack

- Next.js 14 (App Router)
- React 18
- Tailwind CSS 3
- JavaScript
